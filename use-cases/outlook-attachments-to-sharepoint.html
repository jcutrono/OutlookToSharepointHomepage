Attachments are where filing falls apart

It's not the email body that anyone needs three months later — it's the signed contract, the W-9, the invoice PDF, the floor plan attached to the email. And those are exactly the files that get lost. They live inside someone's Outlook, the file name is "scan_001.pdf," and nobody knows it exists until a deadline forces a search.

This add-in extracts attachments at the moment the email arrives or sends, names them according to a template you control, and uploads them straight to the SharePoint library where they actually belong. No browser uploads, no manual renaming, no "Final_v3_REAL.pdf" sitting on a desktop.

File-naming templates you control

Use tokens for sender, subject, date, attachment name, and mapped SharePoint columns:

{MM-DD-YYYY} - {Subject} - {AttachmentName} → 05-02-2026 - Invoice Q1 2026 - report.pdf {SenderName} - {YYYY}-{MM}-{DD} - {AttachmentName} → Acme Corp - 2026-05-02 - statement.pdf

Built for how attachments actually flow

Save attachments only, or with the email

Some teams want only the PDF. Some want the email thread for context. Toggle per save, or set a default in settings.

Dedicated attachment subfolder

Optionally put attachments in a subfolder next to the email, so the document library stays clean and the message file doesn't crowd the view.

Save on Send for outgoing

Sending a contract to a client? Pick the matter folder while composing. The attachment lands there the second you hit Send.

Smart folder suggestions

The add-in learns which SharePoint folder you use for each contact. After a few saves it suggests the right destination automatically.

SharePoint metadata filled in

Map attachment metadata to library columns once. From then on, every saved file has its sender, date, and subject populated.

Duplicate detection

The same invoice forwarded twice won't end up in SharePoint twice. Duplicates are flagged and skipped on save.

Frequently asked questions

How do I save Outlook attachments to SharePoint?

Open the email, open the add-in panel, pick a SharePoint folder, and click Save. Attachments upload directly with the file names from your template.

Can attachments be renamed automatically?

Yes. Configure a naming template once with tokens like {MM-DD-YYYY}, {Subject}, {SenderName}, and {AttachmentName}. Every saved attachment uses it.

Does it work for sent mail?

Yes. Turn on Save on Send and your outgoing attachments file themselves the moment you hit Send.

Can I save attachments without saving the email body?

Yes. Save Format settings let you choose attachments only, email only, or both.

What about duplicate attachments?

The add-in detects duplicates by name and content and skips them on save.

Is there a size limit on attachments?

Limits match SharePoint's own — the add-in passes files straight through, so anything SharePoint accepts uploads cleanly.
